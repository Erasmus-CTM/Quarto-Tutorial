Cheatsheet

A quick reference for everything covered in this section.

YAML — _quarto.yml options

Project

project:
  type: website       # website | book | manuscript | default
  output-dir: docs    # where rendered files go

Execute (global code defaults)

execute:
  echo: true          # show source code
  eval: true          # run the code
  warning: false      # hide warnings
  error: false        # hide errors
  freeze: auto        # auto | true | false

Format

format:
  html:
    toc: true
    toc-depth: 3
    theme: cosmo
    theme:
      - cosmo
      - custom.scss
    code-fold: true
    code-tools: true
    number-sections: false
  pdf:
    toc: true
    documentclass: article
    geometry: margin=2.5cm
  docx:
    toc: true

Website

website:
  title: "My Site"
  navbar:
    left:
      - text: "Home"
        href: index.qmd
      - text: "External"
        href: https://example.com
    right:
      - icon: github
        href: https://github.com/your/repo
  sidebar:
    - id: my-sidebar
      style: docked        # docked | floating
      contents:
        - href: page.qmd
          text: Page Title
        - section: "Group"
          contents:
            - href: sub.qmd
              text: Sub-page

Profiles

profile:
  default: web
  group: [web, print, de, en]

QMD — code chunk options

#| echo: false          # hide source code
#| eval: false          # don't run (show code only)
#| output: false        # hide output
#| warning: false       # hide warnings
#| include: false       # hide everything (no trace in output)
#| code-fold: true      # true | false — collapsible code block
#| code-summary: "Show the code"
#| fig-cap: "My figure"
#| fig-align: center    # left | center | right
#| label: fig-myplot    # for cross-referencing

Cross-reference labels

Type Label prefix Reference
Figure fig- @fig-myplot
Table tbl- @tbl-mydata
Section sec- (on heading) @sec-intro

QMD — conditional content

::: {.content-visible when-profile="de"}
Nur auf Deutsch sichtbar.
:::

::: {.content-hidden when-format="pdf"}
Not in PDF.
:::

QMD — Markdown elements

Callouts

::: {.callout-note}
::: {.callout-tip}
::: {.callout-warning}
::: {.callout-important}
::: {.callout-caution}

::: {.callout-tip collapse="true"}
## Title
Hidden by default.
:::

Shortcodes

{{< include other-file.qmd >}}   ← embed another file
{{< pagebreak >}}                ← force page break (PDF)
{{< kbd Ctrl+C >}}               ← keyboard shortcut styling

Grid layout

::: {.grid}
::: {.g-col-6}
Left column (half width)
:::
::: {.g-col-6}
Right column (half width)
:::
:::

Column widths are in 12ths: g-col-4 = one third, g-col-6 = half, g-col-12 = full.

Extensions — syntax summary

quizdown

```{quizdown}
---
shuffleAnswers: true
---
# Question text?
1. [x] Correct answer
2. [ ] Wrong answer
```

Pyodide

```{pyodide-python}
#| autorun: false
#| readonly: false
print("runs in browser")
```

py-exercise

```{py-exercise}
def my_func(x):
    pass        # student fills this in

## TESTS ##
assert my_func(2) == 4
```

JSXGraph

```jsxgraph
var board = JXG.JSXGraph.initBoard(BOARDID, {
    boundingbox: [-5, 5, 5, -5], axis: true
});
board.create('functiongraph', [Math.sin]);
```

Class Effect
.text-center Centre text
.text-muted Grey text
.text-white White text
.fw-bold Bold
.border Border
.rounded Rounded corners

.bg-primary Primary colour background

.gap-{0-5} Gap between flex items
