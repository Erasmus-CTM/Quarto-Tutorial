Code

Quarto supports three ways to include Python code — all powered by Pyodide (Droettboom et al. 2021), which runs Python directly in the browser.

Output Only

The code runs automatically in the background; only the result is shown:

sin(  0°) = 0.0000
sin( 30°) = 0.5000
sin( 45°) = 0.7071
sin( 60°) = 0.8660
sin( 90°) = 1.0000
sin(180°) = 0.0000
The Code for this Output looked like this: 
```python
#| execute: true # This makes the Code execute during render
#| echo: false # This makes the Code invisible in the final output 
import math

angles = [0, 30, 45, 60, 90, 180]
for deg in angles:
    val = math.sin(math.radians(deg))   
    print(f"sin({deg:3d}°) = {val:.4f}")
```
